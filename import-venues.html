🎾 Import Venues

This page imports 40 venues from the spreadsheet into Firebase Firestore.

Behaviour:

  • New venues — created with all fields.
  • Existing venues — courts count updated if the spreadsheet value is higher.
  • Venues with no change are skipped.

Note: You must be logged in as master on the main app in the same browser session.
